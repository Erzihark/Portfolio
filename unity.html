MyUnity project

A tower defense game

My Unity project was the first experience with a software development project. At the time I only had some coding experience with C++ and Arduino, so I had to learn C# and the Unity API while working in the project, I worked with a partner through github and doing that involved learning project planning and other skills.

One of my roles in the project involved dealing with code optimization and implementation. I specifically dealt with object pooling which is a graphical performance optimization. Also did AI pathfinding with A* algorithm among other things.

Game development is something I like doing in my free time but one day I might try getting in the industry, as of now my focus is in web development. All things considered this game is the reason I fell in love with software development.
